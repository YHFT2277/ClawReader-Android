
```

1. 用数据线连接手机和电脑
2. 把 APK 复制到手机
3. 在手机上点击安装
4. 允许"未知来源"安装
```

## 常见问题

### Q: 推送失败怎么办？
```bash
# 先拉取更新
git pull origin master

# 再推送
git push origin master
```

### Q: 构建失败怎么办？
```
1. 点击失败的 workflow
2. 查看错误日志
3. 截图发给我
```

### Q: 手机安装失败？
```
1. 设置 → 安全 → 允许未知来源
2. 或关闭"纯净模式"
3. 重新安装
```
