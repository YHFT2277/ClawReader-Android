### 任务 8：构建签名 APK

#### 步骤：
```bash
cd D:\Projects\ClawReader-Android\android

# 构建发布版
.\gradlew assembleRelease

# 输出位置
# app/build/outputs/apk/release/app-release-unsigned.apk
```

#### 签名 APK：
```bash
# 生成密钥（只需一次）
keytool -genkey -v -keystore clawreader.keystore -alias clawreader -keyalg RSA -keysize 2048 -validity 10000

# 签名 APK
jarsigner -verbose -sigalg SHA1withRSA -digestalg SHA1 -keystore clawreader.keystore app-release-unsigned.apk clawreader

# 优化 APK
zipalign -v 4 app-release-unsigned.apk ClawReader-v1.0.0.apk
```

**保存位置**：`D:\Projects\ClawReader-Android\上架材料\apk\`
