## 三、截图制作步骤

### 方法一：真机截图（推荐）
1. 在Android手机上安装ClawReader
2. 准备测试文档（中英文各一篇）
3. 按截图内容规划操作
4. 截图后传输到电脑
5. 用PS/在线工具裁剪到1080×1920

### 方法二：模拟器截图
1. 启动Android Studio模拟器
2. 分辨率设置为1080×1920
3. 安装并运行ClawReader
4. 使用模拟器截图功能

### 方法三：设计稿截图
1. 用Figma/Sketch设计界面
2. 导出高清图片
3. 注意：需与实际UI一致

## 四、截图优化建议

### 视觉设计
- **背景**：使用渐变色或纯色背景
- **文字**：添加简短说明文字（底部或侧边）
- **高亮**：用箭头或圆圈标注重点功能
- **设备框**：可添加手机外框装饰（可选）

### 文案规范
- 简洁有力，不超过10个字
- 突出核心价值
- 避免专业术语

### 示例文案
```
截图1：智能文档管理
截图2：沉浸式阅读
截图3：AI智能摘要
截图4：即时问答
截图5：多语言翻译
```
