# ClawReader 手机测试指南

## 测试环境准备

### 你需要准备
1. **安卓手机**（Android 8.0 以上）
2. **USB 数据线**
3. **测试文档**（准备几个不同格式的文件）

### 测试文档建议
| 文件名 | 内容 | 用途 |
|--------|------|------|
| test.txt | 中文文章 | 测试基础阅读 |
| test.md | Markdown格式 | 测试格式渲染 |
| test.json | JSON数据 | 测试代码高亮 |
| test.py | Python代码 | 测试代码阅读 |
| large.txt | >1MB大文件 | 测试性能 |

---

## 安装方法

### 方法一：直接安装（推荐）
```
1. 电脑下载 APK 文件
2. 微信/QQ 发送到手机
3. 手机点击安装
4. 允许"未知来源"
```

### 方法二：ADB 安装（高级）
```bash
# 手机开启开发者模式 + USB调试
# 连接电脑后执行：
adb install ClawReader.apk
```

## 测试流程

### 第一阶段：基础功能（10分钟）

#### 1. 启动测试
- [ ] 应用图标显示正常
- [ ] 启动画面显示
- [ ] 主界面加载完成

#### 2. 文件打开测试
- [ ] 点击"打开文件"按钮
- [ ] 文件选择器正常弹出
- [ ] 能选择内部存储文件
- [ ] 能选择 SD 卡文件

#### 3. 文档显示测试
| 格式 | 测试文件 | 预期结果 |
|------|---------|---------|
| TXT | 中文小说 | 正常显示，无乱码 |
| MD | 说明文档 | 标题、列表渲染 |
| JSON | 配置文件 | 格式化显示 |
| JS/PY | 代码文件 | 语法高亮 |

---

### 第二阶段：AI 功能（15分钟）

#### 1. AI 摘要
```
操作：
1. 打开一篇长文章
2. 点击"摘要"按钮
3. 等待 AI 生成

预期：
- 显示加载状态
- 生成中文摘要
- 摘要内容准确
```

#### 2. AI 翻译
```
操作：
1. 选中一段英文文本
2. 点击"翻译"
3. 选择"中文"

预期：
- 弹出语言选择
- 显示翻译结果
- 翻译准确通顺
```

#### 3. AI 出题
```
操作：
1. 选中一段学习内容
2. 点击"出题"
3. 查看生成的题目

预期：
- 生成选择题
- 选项完整
- 答案正确
```

#### 4. AI 对话
```
操作：
1. 输入问题"这篇文章讲了什么"
2. 发送消息
3. 查看回复

预期：
- 回复与文档相关
- 回复准确有用
```

---

### 第三阶段：边界测试（10分钟）

#### 1. 大文件测试
```
操作：打开 5MB 以上的文本文件
预期：
- 不崩溃
- 加载时间 < 5秒
- 滑动流畅
```

#### 2. 无网络测试
```
操作：关闭 WiFi/移动数据，使用 AI
预期：
- 友好提示"网络不可用"
- 不崩溃
- 本地功能正常
```

#### 3. 快速操作测试
```
操作：快速切换多个功能
预期：
- 不卡顿
- 不崩溃
- 状态正确
```

#### 4. 后台切换测试
```
操作：
1. 打开文档
2. 切换到其他应用
3. 再切回来

预期：
- 文档状态保持
- 不重新加载
```

## 常见问题排查

### 问题1：安装失败
```
原因：未允许未知来源
解决：设置 → 安全 → 允许未知来源安装
```

### 问题2：打开文件失败
```
原因：缺少存储权限
解决：设置 → 应用 → ClawReader → 权限 → 允许存储
```

### 问题3：AI 功能无响应
```
原因：网络问题或 API 限制
解决：
1. 检查网络连接
2. 查看错误提示
3. 稍后重试
```

### 问题4：显示乱码
```
原因：文件编码问题
解决：尝试用 UTF-8 编码保存文件
```
